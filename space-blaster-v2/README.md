# Space Blaster V2 - Enhanced Edition

A fresh, clean version of the space shooter game for continued development.

## Features
- Optimized collision detection
- Performance improvements
- Boss battles
- Multiple enemy types
- Power-ups and upgrades
- Particle effects
- Sound effects

## Files
- `index.html` - Main game file
- `space-shooter.html` - Alternative entry point
- `src/game.js` - Core game logic
- `assets/` - Game assets (sounds, etc.)

## Development
This is a clean copy created for safe development without affecting previous versions.

## How to Play
1. Open `index.html` in a web browser
2. Click "START GAME"
3. Move mouse to control ship
4. Auto-fire is enabled
5. Destroy enemies and survive boss battles!
