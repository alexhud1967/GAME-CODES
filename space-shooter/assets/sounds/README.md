# 🔊 Sound Files for Scary-Cute Space Invaders

Place your sound files in this folder using these exact names:

## 📁 **REQUIRED SOUND FILES:**

- **`shoot.mp3`** (or `shoot.wav`) - Cute shooting sound (plays when holding LMB)
- **`enemy-hit.mp3`** (or `enemy-hit.wav`) - Enemy damage sound  
- **`explosion.mp3`** (or `explosion.wav`) - Enemy death explosion
- **`powerup.mp3`** (or `powerup.wav`) - Powerup collection chime
- **`juggernaut-hit.mp3`** (or `juggernaut-hit.wav`) - Boss damage roar
- **`background.mp3`** (or `background.wav`) - Spooky-cute background music

## 🎵 **SOUND RECOMMENDATIONS:**

### 🎯 **shoot.mp3/wav**
- **Style**: Cute, bell-like "pew pew" sound
- **Duration**: Short (0.1-0.3 seconds)
- **Volume**: Medium-low (will be set to 30% in game)

### 😢 **enemy-hit.mp3/wav**
- **Style**: Adorable "pop" or squeak sound
- **Duration**: Very short (0.1-0.2 seconds)
- **Volume**: Medium (will be set to 40% in game)

### 💥 **explosion.mp3/wav**
- **Style**: Satisfying boom, not too harsh
- **Duration**: Short to medium (0.3-0.8 seconds)
- **Volume**: Medium-high (will be set to 50% in game)

### ⚡ **powerup.mp3/wav**
- **Style**: Happy chime or magical sparkle sound
- **Duration**: Medium (0.5-1.0 seconds)
- **Volume**: Medium-high (will be set to 60% in game)

### 👹 **juggernaut-hit.mp3/wav**
- **Style**: Scary roar or growl (gets more intense as boss weakens)
- **Duration**: Medium (0.5-1.5 seconds)
- **Volume**: High (will be set to 70% in game)

### 🎵 **background.mp3/wav**
- **Style**: Spooky-cute ambient music, should loop seamlessly
- **Duration**: 30 seconds to 2 minutes (will loop automatically)
- **Volume**: Low (will be set to 20% in game)

## 🛠️ **TECHNICAL NOTES:**

- **Both MP3 and WAV formats are supported** - the game will try MP3 first, then WAV
- **Files are loaded with `preload="auto"`** for instant playback
- **All sounds have proper error handling** - missing files won't break the game
- **Background music starts on first user interaction** (browser requirement)
- **Shooting sounds are optional** - only play when holding left mouse button

## 🎮 **TESTING YOUR SOUNDS:**

1. Add your sound files to this folder
2. Open the game in your browser
3. Click the 🔊 button to enable sound
4. Move your mouse to start background music
5. Hold LMB while playing to hear shooting sounds
6. Test all sound effects by playing the game!
