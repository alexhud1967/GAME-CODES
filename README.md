# GAME-CODES
JAVA/HTML GAMES

## Repository Structure
This repository contains individual game projects organized in game-specific folders. Each game can be tagged for easy sorting by platform (linux, steam, itch.io, html5, etc.).

```
GAME-CODES/
├── README.md
├── shared-assets/          # Common assets used across games
│   ├── sprites/           # Shared sprite files
│   ├── sounds/            # Shared audio files
│   ├── fonts/             # Shared font files
│   └── scripts/           # Shared utility scripts
└── [game-name]/           # Individual game folders
    ├── README.md          # Game-specific documentation
    ├── src/               # Source code
    ├── assets/            # Game-specific assets
    └── build/             # Build outputs (if applicable)
```

Use repository tags to categorize games by platform:

- `linux` - Linux native games
- `steam` - Steam-compatible games
- `itch.io` - Itch.io compatible games
